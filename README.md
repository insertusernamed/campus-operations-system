# Campus Scheduler

A full-stack web application for analyzing and optimizing university classroom scheduling. Built as a solo capstone project for COMP 4431/4432 at Lakehead University.

## Overview

Campus Scheduler helps administrators manage physical university resources (classrooms and labs) by detecting scheduling conflicts, analyzing room utilization, and eventually auto-generating optimized schedules.

### Features

- **Conflict Detection** - Validate schedules against room capacity and time slots to prevent double-booking or over-capacity assignments
- **Utilization Analytics** - Identify underused rooms and peak usage periods through summary stats and charts
- **Resource Management** - CRUD interface for courses, instructors, rooms, and time slots
- **Auto-Scheduling** - Constraint-based algorithm to automatically generate valid schedules (planned)
